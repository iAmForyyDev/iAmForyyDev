## 👾 About me

I'm a self-taught **Java developer** with over 5 years of experience building Minecraft plugins. It all started as a kid, trying to understand how my favorite server worked. That curiosity never left.

Since then I've gone from small utility plugins to distributed, high-performance systems capable of handling thousands of concurrent players. When the standard API isn't enough, I go deeper with **NMS** to get things done right. Performance is never an afterthought for me, it's part of how I think from day one.

I write clean, modular and maintainable code. I'm always looking for ways to improve, always learning something new, and I genuinely enjoy collaborating and building things that actually matter to the community.

Outside of code, I'm into old-school salsa and boxing. Both teach the same thing: discipline, patience and the will to keep going.

Currently building <a href="https://mcrankeds.com">**McRankeds**</a>, a ranking platform that connects multiple Minecraft servers under a unified competitive system, and maintaining <a href="https://bytephoria.team">**Bytephoria**</a>, my open-source space where I release free plugins for the community.
